  1. 로그인 기능 (kakao, naver)
  2. 회원가입 기능 (sns로그인 연동)
  3. 공공데이터 기반 예약기능
  4. 예약조회기능 (비회원 조회, 회원 조회)
  5. 검색기능 (지역, 예약날짜/시간, 예약가능 조회)
  6. 상세보기 (naverMap, 공공데이터 기반)
